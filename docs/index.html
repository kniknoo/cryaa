    cryaa

    Ascii Art Library written in Crystal

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     cryaa:
   github: kniknoo/cryaa

    `

    1. Run shards install

    Usage

    require "cryaa"

    The file asciiart.cr can be run as a quick example, include a png file to open
